Materiales

Arduino

Microcontrolador programable util y flexible para el uso de proyectos dentro del campo de la electronica.

Modulos de humedad, temperatura, y bluetooth

Modulos usados en el proyecto para rastrear ubicacion, humedad y temperatura.

Visual Studio Code

Usando componentes de HTML, JavaScript, CSS, y SQL para desarrollar el backend y el frontend del proyecto, asimismo, se usan en conjunción de Arduino IDE y Android Studio para configurar el cliente.

Mapa interactivo

Aplicando la librería abierta de Leaflet, se puede implementar un mapa interactivo que es capaz de demostrar gráficamente los datos relevantes para el usuario.

Presentacion del Proyecto

Alcances

Con el objetivo de entender las aplicaciones de la ingenieria electronica, se considero un proyecto en donde nosotros como estudiantes podamos observar problemas de la vida cotidana, y atraves del desarollo del software, encontrar soluciones respectivas usando una base extensa de herramientas. En nuestro caso, la aplicacion del Arduino, sus modulos, y la creacion de backends y frontends.

Objetivos

Con las herramientas de Android Studio, ArduinoIDE, Visual Code Studio, y sus componentes; se desea lograr poder crear un dispositivo que sea capaz de poder registrar la humedad y la temperatura de una cierta ubicacion, este dispositivo tiene como objetivo ser usado como una herramienta para poder determinar si la humedad y la temperatura del sitio se presenta en los estandares apropiados, y alternativamente, como un ubicador gps que podria ubicar objetos basicos sin funcionalidad GPS.

Funcionamiento

Visual Code Studio

Visual Code Studio fue la herramienta principal al crear la el backend y el frontend de este proyecto, con esta herramienta se posibilito craer el sniffer de la aplicacion, o archivo capaz de recibir paquetes por medio de UDP, y guardarlas a la base de datos. De igual forma se logro crear el servidor por medio de Node.js. El frontend, o lo que se esta observando ahora mismo es un producto de la programación por html y configurado a traves de archivos css.

SQLWorkbench

SQLWorkbench es una herramienta util que es capaz de crear, administrar y hacer operaciones con bases de datos. SQLWorkbench nos deja guardar los datos del arduino en acorde con su tipo de dato y guardalo dentro de una tabla para futura referencia.

Leaflet

Leaflet es una libreria abierta que nos accede a un sistema de cartografia que nos habilita ubicar las coordenadas del arduino, y sus correspondientes datos como la humedad y la temperatura.

Arduino

Es una placa de desarrollo de hardware para construir dispositivos digitales y dispositivos interactivos que puedan detectar y controlar objetos del mundo real.

ArduinoIDE

Es un programa el cual permite cargar un código desde el computador hasta la placa del Arduino, el programa permite programar un código en el lenguaje C++.

Sensor DHT-11 (Humedad y Temperatura)

Este sensor requiere una fuente de alimentación de 3.5V hasta 5V, el rango para la humedad es de 20% hasta el 90% y la temperatura es medible de 0°C a 50°C, la precisión para la humedad es de ±2% y la precisión para la temperatura es de 2°C.

Sensor HC-05 (Bluetooth)

Este sensor nos permite conectar nuestro Arduino de forma inalámbrica con nuestro computador, celular, etc. Esto sirve para la transmisión de datos, recepción de información, entre otras cosas.

Resultados

El arduino existosamente logro rastrear los datos requiridos, y asimismo mandar a traves de bluetooth los datos desde el arduino hasta un celular, y desde ese mismo celular, a traves del desarrollo del software se logra mandar los datos a traves de UDP hacia el servidor en donde reside la pagina. Con estos datos, la pagina guarda los datos en nuestra base de datos y con esos datos logra graficar la informacion atraves de leaflet con sus acompañantes datos. Gracias a la caracteristica de leaflet de marcadores, se hace posible no solo mostrar la ubicacion, pero mostrar dentro de una caja la humedad y la temperatura.

Conclusiones

Hay muchos aplicaciones dentro de la Electronica, y entre esas el desarollo de software es una aplicacion que abre muchas puertas. Con la combinacion de conocimientos electronicos y habilidades para programar en diferentes lenguas de programacion, No existe un limite de lo que se puede y no se puede hacer. Lo mas importante es al momento de inventar un proyecto, se tiene que saber que hacer y como hacerlo con precision y eficiencia para ser un ingeniero efectivo y eficaz.
